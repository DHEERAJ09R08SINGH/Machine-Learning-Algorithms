# 🧠 Machine Learning Algorithms Overview

This repository provides a structured overview of Machine Learning algorithms, categorized into supervised, unsupervised, semi-supervised, and reinforcement learning. It is designed for beginners and interview preparation with simple explanations and visual representations.

---

## 📌 Table of Contents
- Supervised Learning
- Unsupervised Learning
- Semi-Supervised Learning
- Reinforcement Learning

---

## ✅ 1. Supervised Learning
Algorithms learn from labeled data (input + output) and are used for prediction tasks.

### 📈 Regression
- Linear Regression  
- Ridge / Lasso / ElasticNet  
- Polynomial Regression  
- Support Vector Regression (SVR)  
- Decision Tree  
- Random Forest  
- Gradient Boosting  
- XGBoost / LightGBM / CatBoost  

### 🏷️ Classification
- Logistic Regression  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)  
- Decision Tree  
- Random Forest  
- Naive Bayes  
- AdaBoost  
- Neural Networks (ANN, CNN, RNN)  

---

## 🔍 2. Unsupervised Learning
Algorithms work with unlabeled data to find hidden patterns.

### 📊 Clustering
- K-Means  
- Hierarchical Clustering  
- DBSCAN  
- Mean Shift  
- Gaussian Mixture Model (GMM)  

### 📉 Dimensionality Reduction
- PCA  
- LDA  
- t-SNE  
- UMAP  

### 🔎 Association Rules
- Apriori  
- FP-Growth  
- Eclat  

### ⚠️ Anomaly Detection
- Isolation Forest  
- One-Class SVM  
- Local Outlier Factor (LOF)  

---

## ⚖️ 3. Semi-Supervised Learning
Uses both labeled and unlabeled data to improve learning performance.

- Self-Training  
- Label Propagation  
- Label Spreading  
- Semi-Supervised SVM  
- Co-Training  

---

## 🎮 4. Reinforcement Learning
Algorithms learn through interaction using rewards and penalties.

### 🧠 Value-Based
- Q-Learning  
- SARSA  

### 🤖 Deep RL
- DQN  
- Double DQN  
- Dueling DQN  

### 🎯 Policy-Based
- Policy Gradient  
- REINFORCE  

### ⚙️ Actor-Critic
- A2C / A3C  
- PPO  
- DDPG  
- TD3  
- SAC
